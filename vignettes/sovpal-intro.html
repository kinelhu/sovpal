Introduction to sovpal

library(sovpal)
has_ggplot2 <- requireNamespace("ggplot2", quietly = TRUE)

Introduction

The sovpal package provides color palettes derived from Soviet industrial standards and military paint specifications. The palettes span qualitative, sequential, and diverging types, with full ggplot2 integration.

Design Philosophy

The standards-based palettes (gost14202, gost_signal, soviet_military, steppe, hazard) carry fixed hex values traceable to specific documents — GOST paint card numbers, paint manufacturer references, or military paint standards. Modifying these values would alter the historical record.

The constructivist palette is evocative rather than standards-derived; its colors are not fixed in the same sense.

White-background usability is always addressed through the subset, from, and to parameters, not by changing values.

sovpal("gost14202")

#> "#2E7D32" "#C62828" "#5B8DB8" "#F9A825" "#E64A19" "#8E6EAF" "#5D4037" "#78909C"

sovpal("gost14202", n = 4)
#>     water      fire       air       gas 
#> "#2E7D32" "#C62828" "#5B8DB8" "#F9A825"

ramp <- sovpal("steppe", n = 100, type = "continuous")

White-Background Usage

Color scales: automatic

scale_color_sovpal() automatically applies each palette’s lines-safe subset by default, dropping colors that are low-contrast for points and lines on white backgrounds. You don’t need to specify anything:

library(ggplot2)

# Yellow dropped automatically from gost14202 -- just works
ggplot(iris, aes(Sepal.Length, Sepal.Width, colour = Species)) +
  geom_point(size = 2) +
  scale_color_sovpal("gost14202") +
  theme_minimal()

To opt out and use the full palette:

scale_color_sovpal("gost14202", auto_subset = FALSE)

Fill scales: manual when needed

scale_fill_sovpal() uses the full palette. For most palettes this is fine. The one case requiring a manual subset is constructivist, where the cream stop (#E8DFC8) is invisible even as a bar fill:

# Cream excluded manually for fill on white
ggplot(iris, aes(Species, Sepal.Length, fill = Species)) +
  geom_boxplot() +
  scale_fill_sovpal("constructivist",
    subset = c("red", "black", "blue")) +
  theme_minimal()

Using sovpal() directly

When building base R plots or computing colors manually, use the subset and from parameters directly:

# Lines-safe subset from gost14202
sovpal("gost14202", subset = c("water", "fire", "air", "toxic", "alkali", "other"))
#>     water      fire       air     toxic    alkali     other 
#> "#2E7D32" "#C62828" "#5B8DB8" "#E64A19" "#8E6EAF" "#78909C"

# Constructivist without cream
sovpal("constructivist", subset = c("red", "black", "yellow", "blue"))
#>       red     black    yellow      blue 
#> "#CD2500" "#1A1A1A" "#F9A825" "#1565C0"

ggplot2 Integration

library(ggplot2)

df_fluid <- data.frame(
  fluid = factor(names(sovpal("gost14202")), levels = names(sovpal("gost14202"))),
  flow  = c(120, 95, 60, 80, 45, 70, 110, 55)
)

ggplot(df_fluid, aes(fluid, flow, fill = fluid)) +
  geom_col() +
  scale_fill_sovpal("gost14202") +
  labs(title = "GOST 14202-69: Pipeline Colors", x = NULL, y = "Flow rate (L/min)") +
  theme_minimal() +
  theme(legend.position = "none", axis.text.x = element_text(angle = 30, hjust = 1))

# Auto lines-safe subset -- no manual specification needed
ggplot(iris, aes(Sepal.Length, Sepal.Width, colour = Species)) +
  geom_point(size = 2.5) +
  scale_color_sovpal("soviet_military") +
  labs(title = "soviet_military -- auto lines-safe subset") +
  theme_minimal()

# Hazard diverging palette -- risk scores, hazard ratios, survival data
m <- outer(1:10, 1:10, function(x, y) (x - 5.5) * (y - 5.5))
df_hm <- data.frame(
  x = rep(1:10, 10),
  y = rep(1:10, each = 10),
  z = as.vector(m)
)
ggplot(df_hm, aes(x, y, fill = z)) +
  geom_tile() +
  scale_fill_sovpal("hazard", discrete = FALSE) +
  labs(title = "Hazard Palette -- Diverging Risk Heatmap", fill = "Score") +
  theme_minimal() +
  coord_equal()

# Steppe sequential gradient
df_seq <- data.frame(x = factor(1:8), y = c(3, 7, 2, 9, 5, 6, 4, 8))
ggplot(df_seq, aes(x, y, fill = y)) +
  geom_col() +
  scale_fill_sovpal("steppe", discrete = FALSE) +
  labs(title = "Steppe Sequential Palette") +
  theme_minimal() +
  theme(legend.position = "none")

Historical Notes

GOST 14202-69 — Pipeline Identification

GOST 14202-69 (Truboprovody promyshlennykh predpriyatiy. Opoznavatelnaya okraska, preduprezhdayushchie znaki i markirovochnye shchitki) was adopted in 1969 and standardized the color-coding of industrial pipelines across all Soviet enterprises. The standard specified eight base colors corresponding to fluid categories, referenced by paint card number. The gost14202 hex values correspond to those card-indexed samples. The system remains mandatory in post-Soviet industrial facilities operating under inherited Soviet technical regulations.

GOST 12.4.026 — Safety Signage

GOST 12.4.026 standardized workplace safety signage colors across the Soviet Union. The four signal colors encode hierarchical urgency: red (immediate danger, prohibition), yellow (caution, warning zones), green (safe conditions, evacuation routes), and blue (mandatory action, PPE requirements). Post-Soviet revisions preserved all four signal colors with minimal change.

4BO Protective Green

4BO (Zashchitnyy) is the canonical Soviet military vehicle exterior color, adopted in the 1930s and used continuously through WWII and the Cold War. The hex value #51653F is based on Vallejo Model Color 70.609 and corroborated by AK Interactive spectrographic measurements of surviving vehicle examples. It appears in soviet_military (green_4bo) and steppe (stop 2).

Cockpit Turquoise

Soviet aircraft cockpits from the 1940s through the 1970s were finished in a distinctive grey-turquoise used across diverse airframes including the MiG-15, MiG-21, Tu-95, and Su-27. The value #5B8F8C is derived from AK Real Colors RC206 (Interior Turquoise Green), formulated from spectrographic analysis of preserved cockpit interiors.

Constructivist Poster Design

Soviet Constructivism (1920s–1930s) developed a visual language centered on a small set of high-contrast colors: red, black, and white, with yellow and blue as secondary elements. The constructivist palette is evocative of this corpus rather than derived from a specific technical standard. The cream (#E8DFC8) reflects the typical ground color of printed poster reproductions and is low-contrast on white backgrounds.
