## Notes for the reviewer

* The artistic palettes (lissitzky, popova, stepanova, malevich) are
  color-sampled approximations from specific public-domain artworks; the
  sources are recorded in data-raw/sample_artwork.R and described in the
  vignette and NEWS.md. No copyrighted image is bundled with the package; the
  one image still under copyright (Stepanova, EU copyright until 2029) is shown
  in the vignette as swatches with a link to the source rather than reproduced.
* The package vendors no data beyond small public-domain thumbnail images used
  only to build the vignette.
